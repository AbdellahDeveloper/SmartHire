AI Assistant Live Demo

Test the bot instantly — no Microsoft Teams setup required

What is this?
This is a web-based simulation of the assistant that normally runs inside Microsoft Teams.

Important to know:
This demo version runs with a lighter model and simplified environment, so it may not be as powerful or feature-complete as the main application.

Why use it?
It’s a fast and easy way to experiment, test conversations, and see how the chat experience works before using the full system.
